Programa Insiders - Fidelização de Clientes

Contexto: A Mercury Store é uma empresa inglesa fictícia de outlet multimarcas que opera via e-commerce. Em pouco mais de 1 ano de operação, o time de marketing observou que alguns clientes contribuem de forma significativa para o faturamento da empresa.

Desafio: Selecionar os clientes mais valiosos para integrar um programa de fidelidade personalizado, utilizando dados de compra e técnicas de Machine Learning.

Questões de negócio

  • Quem são os clientes elegíveis?
  • Quantos clientes farão parte do grupo?
  • Quais as características dos clientes Insiders?
  • Qual a expectativa de faturamento do grupo?
  • Como incluir ou remover clientes do programa?

Dados utilizados

Faturas de vendas com as colunas: InvoiceNo, StockCode, Description, Quantity, InvoiceDate, UnitPrice, CustomerID, Country.

Premissas adotadas

  • Clientes com devoluções frequentes foram excluídos.
  • Produtos com preço abaixo de £0,04 foram considerados brindes.
  • Códigos de taxas e postagem foram desconsiderados.

Etapas da Solução

  1. Coleta e limpeza de dados: Kaggle + pandas.
  2. Feature engineering: criação de métricas de valor e frequência.
  3. Análise exploratória: uso de Pandas Profiling/YData Profiling.
  4. Modelagem: aplicação de UMAP, PCA, t-SNE para redução de dimensionalidade. Modelos: K-Means, GMM, HC, DBSCAN.
  5. Avaliação: métrica Silhouette Score.
  6. Resultado: escolha de 9 clusters com K-Means (score: 0.585).

Ferramentas Utilizadas

  • Python 3.8 (pandas, seaborn, matplotlib, sklearn)
  • Banco de dados: SQLite, PostgreSQL
  • Cloud AWS: EC2, RDS, S3
  • Power BI para visualização

Resultados de Negócio

  • Insiders representam 58,8% do volume de itens vendidos
  • Contribuem com 53,8% da receita total
  • Média de devoluções ligeiramente superior
  • Mediana de faturamento 781x maior que a base geral

Deploy

Criação de pipeline com Amazon S3, Postgres (RDS), EC2 e agendamento de classificação de novos clientes. Dashboards interativos via Power BI Service.

Conclusão

O programa Insiders foi estruturado com base em dados reais, resultando em segmentação inteligente de clientes e automação do processo de atualização. O modelo está preparado para operar em produção com ganhos comprovados de receita e engajamento.
